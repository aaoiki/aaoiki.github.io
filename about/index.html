Hello

My name is Ahmed [ˈɑːmæd]. It’s derived from the verb that means to praise or to thank in Arabic. A name I found similar to mine is the name Amadeus which means “to love God”.

I’m a programmer based in Cairo, Egypt. I have been working on both the backend as well as the frontend for a while now (although I still don’t know how that started but here we are).

I’m most interested in software engineering principles, how software performs, and building large systems. I’m also fond of linguistics and languages; both programming languages and human natural languages.

Aside from coding (I tend to think I have a life), I like to create things; be it photographs, vector designs, music or whatever project I’m working on right now.

You can find me around the web as @aonemd.
